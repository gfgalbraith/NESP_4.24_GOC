# NESP_4.24_GOC

This repository contains the collated data associated with report for NESP Project 4.24 – Towards assessing the values of reefs in the southern Gulf of Carpentaria. A synthesis of the existing scientific data on the coral reefs in the southern Gulf of Carpentaria.

Report prepared by: Maya Srinivasan, Gemma Galbraith and Damien Burrows

Centre for Tropical Water and Aquatic Ecosystem Research (TropWater), James Cook University

Suggested report citation: Srinivasan, M., Galbraith, G.F. and Burrows, D. (2025) NESP Project 4.24 – Towards assessing the values of reefs in the southern Gulf of Carpentaria. A synthesis of the existing scientific data on the coral reefs in the southern Gulf of Carpentaria. [DOI LINK PLACEHOLDER]

Data have been collated from existing sources and should be cited using original attribution.

**Coral reef fish and benthic data**

Ecological survey data collected by Reef Life Survey Foundation (RLS) were extracted from the following following datasets at <https://portal.aodn.org.au/> And extracted using a bounding box with the following coordinates: -9.26N, -18.1S, 142.29E, 131.66W.

IMOS - National Reef Monitoring Network Sub-Facility - Global reef fish abundance and biomass

IMOS - National Reef Monitoring Network Sub-Facility - Global mobile macroinvertebrate abundance

IMOS - National Reef Monitoring Network Sub-Facility - Survey metadata

IMOS - National Reef Monitoring Network Sub-Facility - Global cryptobenthic fish abundance

IMOS - National Reef Monitoring Network Sub-Facility - Global off-transect species observations

IMOS - National Reef Monitoring Network Sub-Facility - Site information

Benthic data was provided directly by RLS (benthic photo annotation data - see rawdata - RLS_GOC_benthicdata.csv)

Hamish Malcolm provided raw data
